# Exercice : Calculatrice

## Objectif du projet
Créer une petite calculatrice fonctionnelle entièrement en HTML, CSS et JavaScript.  
L’objectif est de manipuler le DOM, les événements, et la logique de calcul, tout en construisant une interface simple et agréable à utiliser.

## Consignes

### 1. Structure (HTML)
- Créez une zone d’affichage qui montrera les chiffres et les résultats.
- Ajoutez les boutons suivants :
    - Les chiffres : 0 à 9
    - Les opérations : +, -, ×, ÷
    - Un bouton =
    - Un bouton C pour tout effacer

### 2. Mise en page (CSS)
- La calculatrice doit être centrée dans la page.
- Les boutons doivent être bien alignés sous forme de grille.
- Appliquez un minimum de style afin d’obtenir un rendu propre et lisible.

### 3. Fonctionnement (JavaScript)
- Lorsqu’on clique sur un bouton, la valeur doit apparaître dans l’affichage.
- L’opération doit être correctement calculée lorsque l’utilisateur clique sur =.
- Le bouton C doit remettre l’affichage à zéro.
- Évitez l’utilisation de `eval()` : construisez vous-même la logique de calcul.

## Bonus (facultatifs)
- Ajouter la gestion du clavier.
- Empêcher les doubles opérateurs (ex : 5 ++ 2).
- Ajouter les décimales.
- Ajouter un mode sombre / clair.
